ТЕМА: ВВЕДЕННЯ В РЕЛЯЦІЙНІ БАЗИ ДАНИХ
Завдання 1:
Встановити SQL Server Express та SQL Server Management Studio.
В якості альтернативи SQL Server, ви можете встановити LocalDB використовуючи
Visual Studio Installer.

Завдання 2:
Створити базу даних однієї із систем (на вибір): університет, ресторан, магазин,
пошта, соціальна мережа і т.д.
Мінімальна кількість таблиць – 4, кожна з яких повинна містити по декілька
стовбців різного роду інформації. Необхідно використати різні типи даних,
первинні ключі і тд.
! Результат завантажити у вигляді знімків екрана зі структурою таблиць.
